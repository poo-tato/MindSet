# 🌙 Mind Archive
> **"생각을 비우고, 마음을 채우는 고요한 기록 공간"**

복잡한 생각이 머릿속을 떠나지 않을 때, 정제되지 않은 마음을 쏟아낼 수 있는 미니멀 다크 모드 메모장입니다. 

## ✨ 주요 기능 (Key Features)
- **🎹 ASMR Typing:** 글을 쓸 때 들리는 타자기 소리로 몰입감 있는 기록 경험 제공.
- **⏳ Sand Mode:** 기록하지 않고 흘려보내고 싶은 생각들을 위한 '휘발성 삭제' 애니메이션.
- **🎵 Lofi BGM:** 집중을 돕는 Lofi 비트를 배경음악으로 바로 재생 가능.
- **💾 Local Storage:** 서버 없이 브라우저 내부에 안전하게 저장되는 프라이빗 아카이브.

## 🛠 기술 스택 (Tech Stack)
- HTML5, CSS3, JavaScript (Vanilla JS)
- Web Audio API (타자기 효과음)
- YouTube IFrame Player API (배경음악)

## 🚀 실행 방법 (Live Demo)
1. 상단 `Settings` > `Pages`에서 배포된 링크를 확인하세요.
2. 혹은 `index.html`을 다운로드하여 브라우저에서 실행하세요.
